Hilfreiche Informationen zum Spielen:

Haus -> Erhöht die Bevölkerungskapazität
Gelbes Feld -> Weizenfeld (Nahrungsproduktion)
Dunkelgrünes Feld -> Wald (Holzproduktion)
Schwarzes Feld -> Steinbruch (Steinproduktion)
Rotes Feld -> Siegeswunder (leitet den Sieg ein)

Controls:
Space(drücken) -> EndTurn (1)
Space(halten)  -> EndTurn (∞)

Zum Bauen eines Gebäudes müssen Sie das gewünschte Konstrukt in dem Rechteck am unternen Bildschirmrand auswählen
(dieses bleibt ausgewählt solange Sie es nicht ändern) und auf ein Sechseck im Spielfeld klicken.

Tipps:
Achten Sie darauf, dass ihnen nie die Nahrung ausgeht. Gold wird für jedes Gebäude benötigt. Also sollten Sie so viel Population wie
möglich besitzen, die Gold produziert. Um die Population zu erhöhen müssen Sie Häuser bauen.

Die Siegesbedingung ist es das Wunder zu bauen(rotes Feld). Jedoch können Sie nach dem Sieg immernoch weiterspielen.

Die Zahl in Klammern hinter der Hauptzahl einer Resource gibt den Resourcenkurs an, also um wie viel sich besagte Resource im
nächsten Zug verändert.
